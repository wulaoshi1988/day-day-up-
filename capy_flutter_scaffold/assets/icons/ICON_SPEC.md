# Icon Asset Specification

## Directory Layout

- app/
- nav/
- plan/
- growth/
- parent/
- status/

## Naming Rules

- lower_snake_case only
- no spaces
- no Chinese characters in filenames
- use SVG for UI icons, PNG for launcher icons
